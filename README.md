  Project Overview

This project focuses on understanding and applying digital literacy concepts through practical tasks. It covers important areas such as digital awareness, online platforms, communication, and cyber safety.
The project helps in developing essential digital skills required for academic and professional growth.


Task 1: Infographic

In this task, an infographic was created to represent the concept of digital literacy. It includes topics such as safe internet usage, email etiquette, digital tools, and the importance of digital literacy.

Folder: task-1-presentation/


Task 2: Digital Portfolio

A digital portfolio was created by setting up profiles on professional platforms like GitHub, LinkedIn, and Kaggle. These platforms help in showcasing skills, networking, and career development.

Folder: task-2-portfolio/

Task 3: Exploring Platforms

This task involved solving a coding problem on HackerRank and creating a quiz using Google Forms. It helped in understanding how digital platforms are used for learning and interaction.
